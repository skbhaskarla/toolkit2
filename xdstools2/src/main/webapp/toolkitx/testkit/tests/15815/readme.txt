Responding Gateway negative tests

<h2>XDSUnknownCommunity Error</h2>
<p>
A GetDocuments query sent to a Responding Gateway with a Home Community Id
that is not known by the Responding Gateway. The error code XDSUnknownCommunity
must be returned.
</p>

<h2>XDSMissingCommunity Error</h2>
<p>
A GetDocuments query sent to a Responding Gateway without a Home Community Id.
The error code XDSMissingCommunity
must be returned.
</p>
