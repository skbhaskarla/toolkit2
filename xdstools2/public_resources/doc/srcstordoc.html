Source Stored Document Help

This tool is for grading the XDS.b_Doc_Source_Stores_Document connectathon test.

The test has an XDS Document Source submitting a single document to a Document Repository which in turn forwards the metadata to a Document Registry. The Document Source is required to log the XDSSubmissionSet.uniqueId to the Gazelle chat window as:

SSID: 1.2.3.4.5

This tool:

  • Issues a GetSubmissionSetAndContents Stored Query (using the SSID) to the Registry.
  • The metadata returned is fully validated.
  • Issues a Retrieve Document Set to the Repository to retrieve the document.
  • The document is validated against the metadata (size, hash, mime type).
