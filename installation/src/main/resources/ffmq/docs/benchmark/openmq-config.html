OpenMQ 4.4u2

Installation steps used for this benchmark :

- Unzipped the default install
- Set up a ConnectionFactory with all default settings
- Set up some test destinations with all limits set to 'unlimited'
- Modified (basedir)/var/mq/instances/imqbroker/props/config.properties
Added the following properties :
  • imq.persist.file.newTxnLog.enabled=true
  • imq.persist.file.sync.enabled=true
