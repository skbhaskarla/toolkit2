Test Engine Documentation

The XD* Test Engine, also known as xdstest, is a testing client built to test servers. It implements the following IHE actors:

  • Document Source
  • Document Consumer
  • Document Repository
  • Initiating Gateway
in such a way that they can be used to test these IHE actors:
  • Document Repository
  • Document Registry
  • Document Recipient
  • Responding Gateway

The documentation is split into the following sections:

Concepts - overview and structure
Environments - the testing environments it fits into
Operation - how to use it
Language Definition - the testplan language definition
